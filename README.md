# yztool

极简的中国研究生招生信息网（研招网，yz.chsi.com.cn）信息抓取工具。利用xpath抓取静态数据。

仅需Python，和request、lxml（xpath库）两个包即可运行

生成json格式数据文件，便于转换为其他数据格式。

目前提供：

1. 硕士目录抓取（master_cat）
  - 1205，图书情报与档案管理/信息资源管理（学术型学位硕士），2023年招生单位信息，json格式
  - 1255，图书情报专业硕士，2023年招生单位信息，json格式
